## 1. 온보딩 플로우

### 테스트 케이스
- [ ] 첫 방문 시 온보딩 페이지로 자동 리다이렉트
- [ ] 3개의 소개 슬라이드 표시 (이모지, 제목, 설명)
- [ ] 진행 상태 인디케이터 작동
- [ ] "다음" 버튼으로 슬라이드 이동
- [ ] "이전" 버튼으로 이전 슬라이드 이동
- [ ] 마지막 단계에서 정보 입력 폼 표시
- [ ] 필수 정보 입력 (닉네임, 관계, 돌봄 기간)
- [ ] 동거 여부 체크박스
- [ ] "시작하기" 버튼 클릭 시 메인 홈으로 이동
- [ ] localStorage에 프로필 정보 저장 확인

### 검증 방법
```javascript
// 브라우저 콘솔에서 확인
localStorage.getItem('careloop_user_profile')
localStorage.getItem('careloop_onboarding_complete')
```
