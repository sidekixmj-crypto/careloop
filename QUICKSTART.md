# CareLoop - 빠른 시작 가이드

## 🚀 5분 안에 시작하기

### 1. 서버 실행

```bash
cd my-app
npm run dev
```

### 2. 브라우저에서 열기

http://localhost:3000

### 3. 온보딩 완료

1. 3개 소개 슬라이드 확인
2. 정보 입력:
   - 닉네임: (원하는 이름)
   - 관계: 배우자/자녀/며느리/사위/형제자매/기타
   - 돌봄 기간: (선택)
   - 동거 여부: (체크박스)
3. "시작하기" 클릭

### 4. 감정 기록하기

1. 홈 화면에서 6개 감정 중 하나 선택
2. 메모 작성 (선택사항)
3. "저장하기" 또는 "건너뛰기"

### 5. AI 상담 체험

1. 하단 "AI상담" 탭 클릭
2. 메시지 입력 예시:
   - "오늘 너무 힘들어요"
   - "어떻게 대화해야 할까요?"
   - "불안해요"
3. AI 응답 확인

### 6. 기록 확인

1. 하단 "기록" 탭 클릭
2. 이번 주 감정 차트 확인
3. 전체 기록 리스트 확인

---

## 📱 주요 기능 빠른 가이드

### 홈 (🏠)
- **감정 기록**: 6개 이모지 중 선택
- **메모 작성**: 오늘의 생각과 느낌 기록
- **빠른 액션**: AI 상담, 기록 보기

### AI 상담 (💬)
- **실시간 채팅**: 고민 상담
- **AI 응답**: 공감 + 실천 가능한 조언
- **히스토리 저장**: 이전 대화 확인 가능

### 기록 (📊)
- **감정 차트**: 최근 7일 감정 변화
- **통계**: 총 기록, 이번 주 기록, 메모 수
- **전체 리스트**: 날짜별 감정 및 메모

### 마이페이지 (👤)
- **프로필**: 나의 정보 확인
- **통계**: 기록한 날 수, 총 기록 수
- **로그아웃**: 데이터 초기화

## 💾 데이터 관리

### 저장 방식
- 브라우저 localStorage 사용
- 자동 저장 (별도 동기화 없음)

### 데이터 확인
브라우저 개발자 도구 → 콘솔에서:

```javascript
// 프로필 확인
JSON.parse(localStorage.getItem('careloop_user_profile'))

// 감정 기록 확인
JSON.parse(localStorage.getItem('careloop_emotion_records'))

// 채팅 히스토리 확인
JSON.parse(localStorage.getItem('careloop_chat_history'))
```

### 데이터 초기화
1. 마이페이지 → 로그아웃
2. 또는 브라우저 콘솔에서:
```javascript
localStorage.clear()
```

## 🐛 문제 해결

### 온보딩이 다시 나타나지 않아요
```javascript
// 콘솔에서 실행
localStorage.removeItem('careloop_onboarding_complete')
// 페이지 새로고침
```

### 데이터가 저장되지 않아요
- 시크릿 모드 사용 중인지 확인
- 브라우저 설정에서 쿠키/저장소 허용 확인

### 서버가 실행되지 않아요
```bash
# 포트 충돌 확인
netstat -ano | findstr :3000

# 또는 다른 포트 사용
PORT=3001 npm run dev
```

### 의존성 설치 오류
```bash
# node_modules 삭제 후 재설치
rm -rf node_modules package-lock.json
npm install
```
